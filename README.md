Pubkon-2014
===========

The example project for my talk on [Pubkon 2014][1]. Take a look at the project on [codepen.io][2]. Thanks to [Maciej Ceg≈Çowski][3] for allowing me to use his great talk from [Beyond Tellerrand conference 2014][4].

Download and use
----------------
To use the complete repository with a local server, follow these steps:
1. Install [Node.js][5]
2. Install [Grunt][6]
3. Clone this repository
   ```sh
   $ git clone https://github.com/Fischaela/Pubkon-2014.git
4. Navigate to the Pubkon-2014 folder
   ```sh
   $ cd Pubkon-2014
5. Install dependencies
   ```sh
   $ npm install
6. Run the local server
   ```sh
   $ grunt serve
7. Open <http://localhost:8000> in your browser to view the example project
